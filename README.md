No contexto deste projeto, eu vou testar uma hipótese relacionada às preferências musicais de duas cidades. Para isso, vou analisar os dados de um serviço de streaming de música online, a fim de validar a hipótese apresentada e comparar o comportamento dos usuários nessas duas cidades.

Isso envolve a análise de dados de um serviço de streaming real para comparar o comportamento dos usuários em Springfield e Shelbyville. O projeto está dividido em três etapas, e cada uma delas possui objetivos específicos.
